# SEVA Arogya Product Overview

SEVA Arogya is a cloud-native voice-enabled clinical prescription system designed for Indian healthcare settings. It transforms doctor-patient consultations into structured, professional prescriptions using AI/ML services.

## Core Functionality

The system converts doctor's voice input into structured prescriptions through:
- Real-time medical transcription with Indian English accent support
- Intelligent entity extraction (medications, symptoms, diagnoses)
- Context-aware medication suggestions
- Multi-language prescription generation (English/Hindi)
- Professional PDF output with standardized formatting

## Key Features

- Voice-to-text transcription using AWS Transcribe Medical
- Medical entity extraction via AWS Comprehend Medical
- Smart medication suggestions based on context
- Secure authentication with AWS Cognito (JWT-based)
- Multi-language support (English/Hindi) via AWS Translate
- Professional PDF prescription generation
- Audit logging for compliance

## Target Users

Primary users are doctors in Indian healthcare settings who need to:
- Reduce prescription writing time (target: 70% reduction)
- Maintain standardized prescription formats
- Support patients in multiple languages
- Ensure secure and compliant record-keeping

## Success Metrics

- 70% reduction in prescription writing time
- 90% transcription accuracy for medical terminology
- 99.5% system uptime
- Sub-2-second transcription response time
